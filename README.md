# 🤟 INDIAN SIGN LANGUAGE TEXT & SPEECH CONVERTER

### 🔊 Real-Time AI Powered Indian Sign Language Translation System

---

> A smart AI-based system that converts  
> **Text + Speech → Indian Sign Language (ISL)**  
> using Computer Vision, Deep Learning, and Real-Time Gesture Detection.

---

# 🌟 PROJECT OVERVIEW

This project is designed to bridge communication gaps between normal users and the deaf or hard-of-hearing community using Indian Sign Language (ISL).

The system can:

- Convert spoken words into sign language
- Convert typed text into sign language
- Detect and recognize ISL gestures in real time
- Use AI and Deep Learning for gesture recognition

---

# ✨ KEY FEATURES

## 🎤 SPEECH TO SIGN LANGUAGE
Converts live voice input into Indian Sign Language output.

---

## ⌨️ TEXT TO SIGN LANGUAGE
Transforms typed text into corresponding ISL gestures.

---

## 📷 REAL-TIME GESTURE DETECTION
Uses webcam and AI models to recognize hand signs instantly.

---

## 🧠 AI & DEEP LEARNING BASED
Powered using:
- YOLOv11
- Computer Vision
- Deep Learning
- Image Processing

---

## ⚡ LIVE DETECTION SYSTEM
- Real-time prediction
- Fast gesture recognition
- High accuracy detection
